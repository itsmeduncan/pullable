# Pullable

Loop over a directory of Git repositories, fetch, and fast-forward merge from
origin.

## Installation

Add this line to your application's Gemfile:

    gem 'pullable'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pullable

## Usage

```ruby
pullable ~/Projects
```
